< lang="en"> Funktionen vs. Methoden

Funktionen vs. Methoden

Sowohl Funktionen als auch Methoden sind Codeblöcke. Während Methoden einem bestimmten Objekt zugeordnet sind, können Funktionen unabhängig aufgerufen werden. Methoden müssen in Punktnotation über ein Objekt aufgerufen werden.

Funktionen können alle Arten von Daten verarbeiten, während Methoden die Daten des Objekts selbst mit dem Keyword this modifizieren.

Variablen vs. Attributen

Variablen und Attribute sind Verweise auf Daten bzw. Werte. Variablen werden mit einem "=" zugewiesen, Attribute mit einem ":".

Außerdem sind Variablen durch ihren Scope (Global, Block oder Function Scope) eingegrenzt. Attribute sind nur mit dem Objekt aufrufbar, dem sie zugeordnet sind.
