Verwendung von Abstrakten Klassen

In JavaScript gibt es kein abstract Keyword, daher wird dies über die Validierung im Konstruktor erzeugt. Dies gilt sowohl für Methoden als auch für Attribute.

new.target ist ein Meta-Attribut, das nur existiert, wenn zuvor der Operator new verwendet wurde. new.target erzeugt eine Referenz auf den Constructor oder die Funktion, mit der new aufgerufen wurde. Bei normalen Funktionen ist es undefined.
