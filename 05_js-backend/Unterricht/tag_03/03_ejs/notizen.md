## Extensions

- EJS language support für Syntax-Highlighting

## EJS Syntax

<% ... %>: Jeglicher JavaScript-Code wird ausgeführt. Wird genutzt für Loops, If Statements (Conditionals)

<%= ... %>: Zeigt JavaScript-Expression in HTML an. Wird genutzt, um dynamischen Content in HTML darzustellen.

<%- ... %>: Ähnlich zu <%= ... %>, aber die JavaScript-Expression wird als reines HTML geparsed. Wird genutzt um HTML Markup darzustellen.

<%# ... %>: Kommentare in EJS. Wird nicht in HTML generiert.
